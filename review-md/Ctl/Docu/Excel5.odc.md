**CtlExcel5**

*CtlExcel5* contains the automation interface for Microsoft Excel 5.0. The objects contained in this module are explained in the corresponding help file (VBA_XL.HLP) which is located on the MS Office CD but is not installed by default.

For more information about automation controllers in BlackBox see the [<u>Ctl Docu</u>](Dev-Man.odc.md).
